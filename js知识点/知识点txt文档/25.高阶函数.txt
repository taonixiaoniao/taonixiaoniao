1、高阶函数是对其他函数进行操作的函数，它接收函数作为参数或将函数作为返回值输出。函数也是一种数据类型，同样可以作为参数，传递给另一个
参数使用。最典型的就是作为回调函数。
	（1）接收函数作为参数
	function fn(a, b, callback) {
            		console.log(a + b);
            		callback && callback();
        	}
        	fn(1, 2, function() {
            		console.log('回调');
        	});
	（2）将函数作为返回值输出
	function fn(){
		return function(){ }
	};
	fn();
