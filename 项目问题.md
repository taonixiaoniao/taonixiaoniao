###### 时间格式转化

从mysql获取到的时间格式是2021-01-01T00:00:00000Z

需要转化成2021-01-01 00:00:00

```js
function rTime(date) {
  var json_date = new Date(date).toJSON();
  return new Date(new Date(json_date) + 8 * 3600 * 1000).toISOString().replace(/T/g, ' ').replace(/\.[\d]{3}Z/, '')
}
```
